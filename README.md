# My Portfolio

Ye mera personal portfolio website hai jo **React.js** aur **Tailwind CSS** se banaya gaya hai.  

## Features

- **Home Section** – Short intro aur welcome message
- **About Section** – Mere baare mai info
- **Skills Section** – Tech stack aur skills
- **Projects Section** – Kuch projects jo maine banaye
- **Contact Section** – Contact form ya contact info
- Fully responsive design
- Navigation using **React Router**

## Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Routing:** React Router
